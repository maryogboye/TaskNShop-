## What TaskNShop Does
**TaskNShop** is a mobile app that connects people who need something bought with trusted local personal shoppers.  
Instead of spending time searching stores, cities, or websites, a user describes the item (or uploads a photo), **TaskNShop** finds a nearby verified shopper, and that shopper goes to the market or store, buys the item, and delivers it ‚Äî while keeping the user updated in real time.
